# Q9 – Smart Campus Information System

A single, menu-driven **Python** application (`main_application.py`) that integrates all
the modules built across the Lab 1–8 sessions into one Smart Campus Information System
dashboard (Lab 9 & 10 mini-project integration).

## Integrated Modules
| Module | Lab |
|--------|-----|
| Student Registration & Grade Evaluation | Lab 1 |
| Course Enrollment Management | Lab 2 |
| Student Records Management | Lab 3 |
| Searching & Sorting Student Data | Lab 4 |
| Fee Calculation using Functions | Lab 5 |
| File-based Academic Record Management (CSV) | Lab 6 |
| Directory Scanning with Exception Handling | Lab 7 |
| Student Performance Analytics (NumPy, Pandas, Matplotlib) | Lab 8 |
| Main System Application Dashboard | Lab 9 & 10 |

## How to Run
```bash
pip install numpy pandas matplotlib
python3 main_application.py
```
